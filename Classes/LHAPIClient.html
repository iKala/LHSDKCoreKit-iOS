LHAPIClient

public class LHAPIClient: NSObject, LHURLSessionProtocol, LHURLRequestProtocol, LHURLResponseProtocol

Base class for LIVEhouse.in API.

LHAPIClient provided the basic functionality of an API client, and preserve the flexibility to extend the API client. With this class, we constructed serveral API client modules for different groups of API endpoints.
